Core Concepts

  • Truth Scores: Measuring claim accuracy based on evidence
  • Linkage Scores: Evaluating logical connections between arguments
  • Importance Weights: Filtering out technically true but irrelevant claims
  • Semantic Clustering: Preventing duplicate arguments from inflating scores
