# Idea Stock Exchange

**Wikipedia for Policy Debates**: A platform where arguments are scored by evidence, not popularity.

## What is this?

The Idea Stock Exchange applies Google's PageRank algorithm to ideas. Just as PageRank scores websites based on the network of links, we score arguments based on the network of evidence.

### Core Innovation: ReasonRank

Every argument gets a quantified score based on:
- **Truth Score** (0-100%): Is this claim actually true?
- **Linkage Score** (-100% to +100%): Does it prove your point?
- **Importance Weight** (0.0-1.0): Does this actually matter?

**Argument Strength = Truth × Linkage × Importance**

When evidence changes, scores update automatically throughout the entire network. The truth propagates itself.

## Why This Matters

Traditional online debates are broken:
- Arguments scatter across threads in chronological chaos
- The same debunked claims get repeated endlessly
- Weak reasoning hides behind confident rhetoric
- Important evidence gets buried while soundbites go viral

We've built a system where being *persuasive* beats being *correct*. The Idea Stock Exchange changes that.

## Key Features

- **One Page Per Topic**: Each claim gets a canonical page, not scattered fragments
- **Logical Structure**: Arguments organized by evidence chains, not chronology
- **Dynamic Scoring**: When foundation evidence changes, all dependent conclusions update automatically
- **Semantic Clustering**: Similar arguments merge automatically—you can't inflate scores by restating points
- **Quality Control**: Three layers (semantic clustering, community validation, expert review)
